# Guide de Contribution — {R3}

Merci de l'intérêt que vous portez au Manifeste {R3}.

## Langue des contributions

**Toutes les contributions doivent être rédigées en français.**

Le manifeste est un document politique ancré dans le débat public français. La langue française est la condition de cohérence éditoriale et de traçabilité des discussions.

> La version anglaise (`en/`) est mise à disposition à titre informatif uniquement. Elle n'accepte pas de contributions directes — les modifications de la traduction sont effectuées par l'équipe éditoriale à partir du texte français validé.

---

## Comment proposer une modification ?

Nous n'utilisons pas les Pull Requests GitHub pour modifier le texte du manifeste. Le processus est le suivant :

1. **Demandez l'accès** au document de travail collaboratif en écrivant à [achille@naffati.com](mailto:achille@naffati.com).
2. **Proposez vos modifications** en "Mode Suggestion" dans le document.
3. **Argumentez votre proposition** dans les commentaires.
4. **La "Règle des 3"** s'applique : une modification est intégrée si elle reçoit le soutien explicite (👍) d'au moins 3 membres du comité de lecture.
5. Une fois validé, le texte est exporté en PDF et publié dans ce dépôt.

---

## Ce dépôt GitHub

Ce dépôt sert à la **publication et à la diffusion** des versions validées, pas à la rédaction collaborative directe. Les issues GitHub peuvent être utilisées pour signaler des erreurs factuelles ou des problèmes de mise en forme.

---

## Structure du dépôt

| Dossier | Contenu | Contributions |
|---------|---------|---------------|
| `fr/` | Version française (source) | ✅ Oui, via le processus décrit ci-dessus |
| `en/` | Version anglaise (traduction) | ❌ Non — lecture seule |
